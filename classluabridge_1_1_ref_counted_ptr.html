Detailed Description

template<class T>
class luabridge::RefCountedPtr< T >

A reference counted smart pointer.

The api is compatible with boost::RefCountedPtr and std::RefCountedPtr, in the sense that it implements a strict subset of the functionality.

This implementation uses a hash table to look up the reference count associated with a particular pointer.
