Detailed Description

template<class CounterType>
class luabridge::RefCountedObjectType< CounterType >

Adds reference-counting to an object.

To add reference-counting to a class, derive it from this class, and use the RefCountedObjectPtr class to point to it.

e.g.

class MyClass : public RefCountedObjectType
{
void foo();
// This is a neat way of declaring a typedef for a pointer class,
// rather than typing out the full templated name each time..
typedef RefCountedObjectPtr<MyClass> Ptr;
};
MyClass::Ptr p = new MyClass();
MyClass::Ptr p2 = p;
p = 0;
p2->foo();
